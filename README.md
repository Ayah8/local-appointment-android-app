# Local Appointment Manager (Android)

## Overview
A personal Android portfolio project built to practice Kotlin and modern Android development by creating a simple local appointment management app.

The app allows users to create, view, edit, and delete appointments stored locally on the device.

## Features
- Create and manage appointments
- View upcoming appointments
- Edit and delete existing entries
- Offline-first (local storage)

## Tech Stack
- Kotlin
- Android SDK
- Jetpack Compose
- MVVM architecture
- Room (local database)

## What I Learned
- Structuring an Android app using MVVM
- Building UI with Jetpack Compose
- Managing state and data flow
- Working with local persistence

## Notes
This project is part of my learning portfolio and is not published on the Google Play Store.
