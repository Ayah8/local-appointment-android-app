## Overview
A personal Android portfolio project built to practice building clean, offline-first apps with Kotlin and modern Android development practices.

This app lets users create, view, edit, and delete appointments stored locally on the device.

## Features
- Add and manage appointments
- View upcoming appointments in a list
- Edit and delete appointments
- Offline-first (no backend)
